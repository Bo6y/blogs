作为开发者，有两个顶级编辑器，即使你不用，你也一定要知道它们是什么——
·编辑器之神——Vim
·神的编辑器——Emacs
所以本文就来介绍vim以及它的常用命令，Vim是从vi发展出来的一个文本编辑器。代码补全、编译及错误跳转等方便编程的功能特别丰富，在程序员中被广泛使用

此状态下敲击键盘动作会被Vim识别为命令，而非输入字符
