2016-07-10
CSS
CSS

原理

CSS盒模型

  1. 盒模型 = Margin + Border + Padding + Content
  2. 上下左右边框交界处出呈现平滑的斜线. 利用这个特点, 通过设置不同的上下左右边框宽度或者颜色可以得到小三角, 小梯形等
  3. 调整宽度大小可以调节三角形形状
